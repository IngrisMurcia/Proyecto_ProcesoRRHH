Aplicación desarrollada en C++ orientada a la gestión de procesos administrativos del área de Recursos Humanos. El sistema permite registrar, consultar y administrar información relacionada con empleados, candidatos, nómina y procesos internos, utilizando archivos .txt como almacenamiento de datos.

Funcionalidades principales:
  •	Registro y gestión de Candidatos
  •	Control de Citas programadas
  •	Gestión de Acciones disciplinarias
  •	Registro de Despidos justificados
  •	Control de Empleados promovidos
  •	Generación de Planilla
  •	Cálculo de Liquidaciones
  •	Gestión de Reembolsos
  •	Programación de Capacitaciones
  •	Registro de Solicitudes de limpieza
  •	Base de datos simulada mediante archivos de texto.

Estructura del Proyecto
El proyecto incluye:
•	ProcesoRRHH.cpp – Archivo principal con la lógica del sistema
•	database.txt – Archivo general de almacenamiento de datos
•	Archivos de registros específicos: Candidatos.txt, CitasProgramadas.txt, AccionesDisciplinarias.txt, EmpleadosDespidoJustificado.txt, empleadospromovidos.txt, planilla.txt, liquidacion.txt, solicitud_reembolso.txt, programacioncapacitaciones.txt, solicitudes_limpieza.txt, calculototal.txt

Programas utilizadas:
  •	C++
  •	Programación estructurada
  •	Manejo de archivos (fstream)
  •	Almacenamiento en archivos de texto (.txt)

Cómo ejecutar el proyecto
  1.	Compilar el archivo: g++ ProcesoRRHH.cpp -o ProcesoRRHH
  2.	Ejecutar: ./ProcesoRRHH
  (En Windows: ProcesoRRHH.exe)
